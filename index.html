The Rules

  1. A random number will be displayed in the "Desired", box.
  2. Of the 4 Crystals provided, each has a static numeric value associated with it, lasting the duration of a single game play. You will see the value of the clicked crystal by looking in the "Achieved"
  3. The goal of the Game is to precisely match the value Desired, box to the "Achieved", box.
